# Es-Virtualizzazione-MaximePaci

## Obiettivo del Progetto
Questo repository contiene la consegna per l'esercitazione pratica sulle infrastrutture IT. Il progetto documenta i concetti fondamentali trattati, in particolare:
- I principi della **virtualizzazione** e il ruolo degli Ipervisori (con focus sugli Ipervisori di Tipo 2 come VirtualBox).
- L'importanza di **GitHub** nel flusso di lavoro di un informatico per il backup, il versionamento del codice e la creazione di un portfolio digitale.
